核心功能

OmniMemo 提供完整的会议理解与摘要解决方案

🎙️

多模态输入

支持会议录音、视频、截图、白板照片等多种输入方式，全面捕捉会议内容

🧠

跨模态理解

MiMo-Omni 原生统一处理音频+视频+图像+文本，无需拼接多个 API pipeline

📝

结构化纪要

自动提取决策点、行动项、争议焦点，分层输出（摘要→议题→行动项）

🔊

语音摘要

MiMo-TTS 生成 3 分钟语音回顾 1 小时会议，通勤路上也能掌握要点

🤖

Agent 调度

根据会议类型（周报/技术评审/客户沟通）自动选择分析模板

💬

多轮追问

支持会议内容的深度交互，"上次讨论的 API 方案最终定了没？"

系统架构

基于 MiMo 全栈模型的完整处理流程

┌──────────────────────────────────────────────────────────────┐
│                        OmniMemo                              │
├──────────────────────────────────────────────────────────────┤
│  用户上传                                                    │
│  (录音/视频/截图/白板)                                        │
│       ↓                                                      │
│  MiMo-Omni (跨模态理解)                                      │
│  原生统一处理 音频+视频+图像+文本                              │
│       ↓                                                      │
│  MiMo-Pro (深度推理 Agent)                                    │
│  发言人识别 → 决策提取 → 行动项生成 → 争议标注                 │
│       ↓                                                      │
│  MiMo-TTS (语音播报)                                         │
│  结构化纪要 → 自然流畅的中文/英文语音摘要                       │
│       ↓                                                      │
│  输出：结构化纪要 + 3分钟语音摘要 + 交互式追问                  │
└──────────────────────────────────────────────────────────────┘
